# Sotka po Rus

Telegram-бот для подготовки к ЕГЭ по русскому языку.

Проект представляет собой интерактивный тренажер с заданиями из банка ФИПИ, статистикой прогресса, премиум-функциями, объяснениями к ошибкам, реферальной системой и административными инструментами.

## Возможности

- Тренировка заданий ЕГЭ по русскому языку:
  - №4 — ударения
  - №9 — гласные в корне
  - №10 — приставки
  - №11 — суффиксы
  - №12 — глаголы
  - №15 — Н и НН
- Блиц-формат решения заданий.
- Проверка ответов в реальном времени.
- Подсчет серии правильных ответов.
- Таблица лидеров по каждому типу задания.
- Недельная и общая статистика пользователя.
- Генерация PDF-отчета со статистикой.
- Премиум-доступ:
  - объяснения к ошибкам;
  - расширенная статистика;
  - дополнительные функции для отслеживания прогресса.
- Оплата премиума через YooKassa.
- Реферальная система для получения бесплатного премиума.
- Административные команды.
- Middleware для антифлуда, логирования ошибок и отслеживания активности пользователей.
