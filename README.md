📘 Egeshnik Bot v1.0
Egeshnik Bot — Telegram-бот для подготовки к ЕГЭ по русскому языку. 
Пользователь выбирает номер задания и отвечает на вопросы в формате теста. 
Бот отслеживает серию правильных ответов и помогает закрепить материал.

🚀 Возможности в версии 1.0
📌 Выбор задания из списка (поддержка заданий №4, 9–15)

📖 Отображение вопроса с вариантами ответов

✅ Проверка правильности

🔁 Кнопка "Начать заново" при неправильном ответе

🏠 Кнопка "Меню" для возврата к главному экрану

🔥 Подсчёт стрика (правильных подряд)
